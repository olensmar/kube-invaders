# Kube Invaders

A simple space-invaders game inspired by the Testkube.io color palette.

## Features

- Classic space-invaders gameplay
- Beautiful color scheme matching Testkube.io
- Smooth animations and gameplay
- Score tracking
- Win condition when all enemies are destroyed

## How to Play

1. Open `index.html` in your web browser
2. Use the following controls:
   - **Arrow Keys** or **A/D** to move left/right
   - **Spacebar** to shoot
3. Destroy all enemies to win!
4. Avoid letting enemies reach the bottom

## Game Mechanics

- Enemies move horizontally and drop down when they hit the edges
- Enemy speed increases as you destroy more enemies
- Each enemy destroyed gives you 10 points
- The game ends when all enemies are destroyed (win) or if enemies reach your position

Enjoy the game!

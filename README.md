# Kube Invaders

A simple space-invaders game inspired by the Testkube.io color palette.

## Features

- Classic space-invaders gameplay
- Beautiful color scheme matching Testkube.io
- Smooth animations and gameplay
- Score tracking
- Win condition when all enemies are destroyed

## How to Play

1. Open `index.html` in your web browser
2. Use the following controls:
   - **Arrow Keys** or **A/D** to move left/right
   - **Spacebar** to shoot
3. Destroy all enemies to win!
4. Avoid letting enemies reach the bottom

## Game Mechanics

- Enemies move horizontally and drop down when hitting edges
- Regular purple enemies: 1 hit = 10 points
- Red (special) enemies: 5 hits = 50 points + piercing bullets for 5 seconds per kill
- Killing red enemies increases your speed and activates piercing bullets (super-piercing after 5 seconds)
- Level 5+: Killed red enemies fall as fire bombs
- Level 10+: All red enemies release fire bombs when one is killed
- Level 15: Boss fight! Massive enemy needs 80 hits, drops fire bombs at half health, speeds up as it drops
- Complete levels to earn extra lives and progress to the boss
- Victory at level 15: defeat the boss to save the world!

Enjoy the game!
